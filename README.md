# Valentine

## โปรเจกต์นี้คืออะไร?
นี่คือโปรเจกต์ที่ใช้สำหรับทดสอบการทำงานร่วมกับ GitHub ผ่าน Manus และแสดงข้อความต้อนรับพร้อม ASCII Art รูปหัวใจสำหรับวันวาเลนไทน์

## การแก้ไขโดย Manus
- ทดสอบการอ่านไฟล์ (Read)
- ทดสอบการเขียนไฟล์ (Write/Edit)
- ทดสอบการ Commit และ Push ข้อมูลกลับไปยัง GitHub
- เพิ่มฟังก์ชันการแสดง ASCII Art รูปหัวใจใน `hello_manus.py`

## วิธีการใช้งาน
หากต้องการรันสคริปต์ `hello_manus.py` เพื่อดูข้อความต้อนรับและ ASCII Art รูปหัวใจ สามารถทำได้โดยใช้คำสั่ง:

```bash
python3 hello_manus.py
```

*แก้ไขเมื่อ: 14 กุมภาพันธ์ 2026*
